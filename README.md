# B2B Landing Page — High Performance Frontend

Маркетинговый B2B-лендинг с упором на производительность, SEO и чистую архитектуру фронтенда.  
Проект задуман как **переиспользуемый шаблон** для коммерческих лендингов.

## Ключевая идея проекта

Сделать максимально быстрый, доступный и контролируемый лендинг без фреймворков,  
с ручным управлением рендерингом, стилями и ассетами.

## CSS: Critical / Async split

### Зачем
- ускорить First Paint / LCP
- избежать блокировки рендера
- полный контроль загрузки

### Как работает
Происходит сборка проекта через webpack, в результате которой появляется два минифицированных файла: index-async.min.css и index-critical.min.css.  
Ниже - то, что находится внутри каждого файла.
- Критические стили (index-critical.min.css), содержат в себе:
  - reset
  - layout
  - variables
  - base
  - критические шрифты и стили hero-блока
  - index-critical.css как entry-point
- Асинхронно подгружаемые стили (index-async.min.css), подгружаются неблокирующе:
  - остальные шрифты страницы и стили секций, ниже hero
  - index-async.css как entry-point

## JavaScript

### Подход
  - Vanilla JS (ES modules)
  - 1 entry point (index.js)
  - Компонентная структура без фреймворков

### Реализовано
  - Модальная система:
   • focus trap
   • возврат фокуса
   • ESC / TAB
   • блокировка скролла
  - Формы:
   • HTML5-валидация
   • submit → success modal
   • очистка инпутов без перезагрузки
  - Анимации:
   • CSS-based
   • минимальный JS (одна легковесная библиотека keen-carousel для анимации карусели отзывов, остальное - реализовано самостоятельно с помощью стилей css)
